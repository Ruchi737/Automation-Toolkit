#  Automation Toolkit

A Python-based automation toolkit designed to automate common repetitive tasks and improve productivity. This project combines multiple automation utilities into a single menu-driven application, allowing users to organize files, rename files in bulk, and extract information from websites.

##  Project Overview

Automation Toolkit was built to explore practical applications of Python programming and automation. The project demonstrates file handling, folder management, web scraping, modular programming, and the use of Python libraries to automate everyday tasks.

## Features

###  File Organizer

* Automatically scans a selected folder.
* Identifies files based on their extensions.
* Creates separate folders for different file types.
* Moves files into their respective folders automatically.

###  Bulk File Renamer

* Renames multiple files at once.
* Allows users to provide a custom prefix.
* Preserves original file extensions.
* Useful for organizing large collections of files.

###  Website Scraper

* Connects to a website using its URL.
* Extracts webpage titles and H1 headings.
* Uses BeautifulSoup for HTML parsing.
* Demonstrates basic web scraping concepts.

## Technologies Used

* Python
* os Module
* shutil Module
* requests
* BeautifulSoup (bs4)
